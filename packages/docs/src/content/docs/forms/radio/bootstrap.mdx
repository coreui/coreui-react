## How to use React Bootstrap Radio component

Use CoreUI’s `<CFormCheck type="radio">` component to create Bootstrap-styled radio buttons for selecting one option from many. These components are built to be accessible, customizable, and responsive.

### Default stacked radios

By default, multiple radio buttons rendered as siblings will be stacked vertically with proper spacing.

<Example code={RadioExampleRaw} name="RadioExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
  <RadioExample client:only="react" />
</Example>

### Disabled state

Add the `disabled` prop to disable a radio button. The associated label will also receive a muted style to reflect its state.

<Example code={RadioDisabledExampleRaw} name="RadioDisabledExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
  <RadioDisabledExample client:only="react" />
</Example>

### Inline radios

Display radio buttons in a horizontal row by adding the `inline` prop to each `<CFormCheck type="radio">`.

<Example code={RadioInlineExampleRaw} name="RadioInlineExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
  <RadioInlineExample client:only="react" />
</Example>

### Reverse layout

Use the `reverse` prop to position the label text on the opposite side of the radio input.

<Example code={RadioReverseExampleRaw} name="RadioReverseExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
  <RadioReverseExample client:only="react" />
</Example>

### Radios without labels

You can omit visual labels, but make sure to provide accessible names using `aria-label` for screen readers.

<Example code={RadioWithoutLabelsExampleRaw} name="RadioWithoutLabelsExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
  <RadioWithoutLabelsExample client:only="react" />
</Example>

### Radio toggle buttons

Turn your radios into button-like toggles using the `button` prop. These can also be grouped using a button group component.

<Example code={RadioToggleButtonsExampleRaw} name="RadioToggleButtonsExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
  <RadioToggleButtonsExample client:only="react" />
</Example>

#### Outlined toggle styles

You can apply outlined styles to toggle radios using Bootstrap's button variant system.

<Example code={RadioToggleButtonsOutlinedStylesExampleRaw} name="RadioToggleButtonsOutlinedStylesExample" componentName="React Bootstrap Radio" class="theme-bootstrap">
