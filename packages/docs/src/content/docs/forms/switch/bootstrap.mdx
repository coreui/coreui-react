## How to use React Bootstrap Switch component

Use CoreUI’s `<CFormSwitch>` to create cross-browser, Bootstrap-styled toggle switches that represent binary states such as on/off, enabled/disabled, or true/false.

### Basic usage

A standard switch uses the `type="switch"` option on the `<CFormSwitch>` component.

<Example code={FormSwitchExampleRaw} name="FormSwitchExample" componentName="React Bootstrap Switch" class="theme-bootstrap">
  <FormSwitchExample client:only="react" />
</Example>

### Sizing

Use the `size` prop to adjust the visual scale of the switch. Available sizes include `"lg"` and `"xl"`.

<Example code={FormSwitchSizingExampleRaw} name="FormSwitchSizingExample" componentName="React Bootstrap Switch" class="theme-bootstrap">
  <FormSwitchSizingExample client:only="react" />
</Example>

### Reverse layout

Add the `reverse` prop to place the switch control after the label.

<Example code={FormSwitchReverseExampleRaw} name="FormSwitchReverseExample" componentName="React Bootstrap Switch" class="theme-bootstrap">
  <FormSwitchReverseExample client:only="react" />
</Example>
