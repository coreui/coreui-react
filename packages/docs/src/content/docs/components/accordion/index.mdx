## How to use React Accordion component.

Click the accordions below to expand/collapse the accordion content.

<Example code={AccordionExampleRaw} lang="tsx" name="AccordionExample" componentName="React Accordion">
  <AccordionExample client:only="react" />
</Example>

### Flush

Add `flush` to remove the default `background-color`, some borders, and some rounded corners to render accordions edge-to-edge with their parent container.

<Example code={AccordionFlushExampleRaw} lang="tsx" name="AccordionFlushExample" componentName="React Accordion">
  <AccordionFlushExample client:only="react" />
</Example>

### Always open

Add `alwaysOpen` property to make react accordion items stay open when another item is opened.
