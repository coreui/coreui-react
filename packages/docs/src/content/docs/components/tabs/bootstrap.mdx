## How to use React Bootstrap Tabs component

Use CoreUI's React tabs styled with Bootstrap to organize content into multiple sections that users can switch between using tabs or pills.

### Tabs style

The default `variant="tabs"` creates traditional Bootstrap-style tabbed navigation.

<Example code={TabsExampleRaw} name="TabsExample" componentName="React Bootstrap Tabs" class="theme-bootstrap">
  <TabsExample client:only="react" />
</Example>

### Unstyled

Without the `variant` prop, tabs render with minimal default styles.

<Example code={TabsUnstyledExampleRaw} name="TabsUnstyledExample" componentName="React Bootstrap Tabs" class="theme-bootstrap">
  <TabsUnstyledExample client:only="react" />
</Example>

### Pills style

Use `variant="pills"` to display tabs styled like Bootstrap pills.

<Example code={TabsPillsExampleRaw} name="TabsPillsExample" componentName="React Bootstrap Tabs" class="theme-bootstrap">
  <TabsPillsExample client:only="react" />
</Example>

### Underline style

Use `variant="underline"` to show a bottom border underline beneath the active tab.

<Example code={TabsUnderlineExampleRaw} name="TabsUnderlineExample" componentName="React Bootstrap Tabs" class="theme-bootstrap">
  <TabsUnderlineExample client:only="react" />
</Example>

### Underline border

Use `variant="underline-border"` to show a full underline across the width of the tab.

<Example code={TabsUnderlineBorderExampleRaw} name="TabsUnderlineBorderExample" componentName="React Bootstrap Tabs" class="theme-bootstrap">
  <TabsUnderlineBorderExample client:only="react" />
</Example>

### Fill and justify

Use `layout="fill"` or `layout="justified"` to control how tabs stretch to fill the horizontal space.

<Example code={TabsUnstyledFillAndJustifyExampleRaw} name="TabsUnstyledFillAndJustifyExample" componentName="React Bootstrap Tabs" class="theme-bootstrap">
  <TabsUnstyledFillAndJustifyExample client:only="react" />
</Example>

### Controlled tabs

Use `activeItemKey` and `onChange` to fully control the active tab from your component state.

## Accessibility

CoreUI's Tabs component automatically manages `aria-` and `role="..."` attributes to match Bootstrap and WAI-ARIA accessibility guidelines for tabs.
