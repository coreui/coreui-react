## How to use React Bootstrap Badge component

React Bootstrap Badges scale based on their parent using `em` units, making them easy to embed inside headings, buttons, or navigation items.

<Example code={BadgeExampleRaw} name="BadgeExample" componentName="React Bootstrap Badge" class="theme-bootstrap">
  <BadgeExample client:only="react" />
</Example>

You can also use badges inside buttons and links to display counters or notifications.

<Example code={Badge2ExampleRaw} name="Badge2Example" componentName="React Bootstrap Badge" class="theme-bootstrap">
  <Badge2Example client:only="react" />
</Example>

Keep in mind that depending on usage, badges might be challenging for screen reader users. Add visually hidden helper text when needed to improve accessibility.

<Example code={Badge3ExampleRaw} name="Badge3Example" componentName="React Bootstrap Badge" class="theme-bootstrap">
  <Badge3Example client:only="react" />
</Example>
