| Property | Description | Type | Default |
| --- | --- | --- | --- |
| **className** | A string of all className you want applied to the base component. | `string` | - |
| **component** | Component used for the root node. Either a string to use a HTML element or a component. | `string | ComponentClass<any, any> | FunctionComponent<any>` | 'img' |
| **orientation** | Optionally orientate the image to the top, bottom, or make it overlaid across the card. | `"top" | "bottom"` | - |
